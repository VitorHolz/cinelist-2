Descrição do Projeto: aplicativo para catalogar filmes. O programa permite a criação de listas de watchlist ou de filmes já assistidos e de dar uma nota para eles
(baseado no https://letterboxd.com/)

Instrução:

o código está na branch "master"

npm install -> baixar as dependências do projeto

substituir o "localhost" no código pelo ip do seu computador (tabs/index.tsx - linha 19 | tabs/filme/[id].tsx - linha 68 | tabs/search.tsx - linha 22

executar ' npx json-server --watch db.json ' na pasta do projeto
